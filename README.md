# Bulk SVG to High-Resolution PNG Converter 🎨

A lightweight, high-performance Python script designed for developers, designers, and webmasters to batch-convert scalable vector graphics (`SVG`) into high-resolution, production-ready `PNG` images with custom DPI control.

## ✨ Features
- **High-Res Rendering:** Supports custom DPI configuration (default set to 300 DPI for print and crisp digital layouts).
- **Batch Processing:** Automatically scans and converts an entire folder of SVG files in seconds.
- **Transparency Support:** Perfect handling of alpha channels and transparent backgrounds.
- **Clean Project Architecture:** Pre-configured with safety defaults to prevent tracking local caches.

## 🚀 Quick Start Guide

### 1. Prerequisites
Make sure you have Python installed, then install the required `cairosvg` engine:
```bash
pip install cairosvg

2. How to Use

    Clone this repository or download the script.

    Create a folder named input_svgs in the same directory and paste your SVG vectors there.

    Run the script:

Bash

python svg_converter.py

    Find your high-quality, high-res images inside the automatically created output_pngs folder!
